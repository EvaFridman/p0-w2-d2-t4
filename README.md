# p0-w2-d2-t4 · Карточки, выровненные по внутренним линиям

> Тип: research (на лекции не давали — ищешь сама) · Репозиторий: один отдельный

## Задача

Ряд карточек «заголовок — текст — кнопка». Заголовки разной длины, текста в карточках разное
количество — и кнопки оказываются на разной высоте, ряд выглядит неряшливо. Обычный grid
выравнивает только прямых детей контейнера, до содержимого карточек он не дотягивается. Есть
механизм, который позволяет внутренним строкам карточек выровняться по общим линиям ряда — на
лекции его не было, разбираешься сама.

```
ряд карточек: заголовок / текст / кнопка

без subgrid:  кнопки на разной высоте (текст разный)
  [загол.]   [загол. в 2  ]   [загол.]
  [текст ]   [строки      ]   [текст ]
  [кнопка]   [текст       ]   [кнопка]
             [кнопка ниже!]

с subgrid: внутренние строки карточек выровнены
ищешь сама: как карточка наследует строки родителя
```

## Что нужно сделать

Сделать ряд из 3–4 карточек, в каждой три части: заголовок, описание разной длины, кнопка
внизу. Добиться, чтобы во всех карточках заголовки, тексты и кнопки стояли на общих линиях —
кнопки на одном уровне независимо от длины текста выше. Без JS и без фиксированных высот.

## Куда копать

- почему обычная вложенная сетка внутри карточки не выравнивается с соседними карточками;
- что такое `subgrid` и как карточка может наследовать строки родительского грида вместо того,
  чтобы заводить свои;
- что для этого карточка должна сама быть grid-контейнером и занимать нужное число строк
  родителя (`grid-row: span N`).

## Критерии приёмки

* Заголовки, тексты и кнопки выровнены по общим линиям во всех карточках; кнопки на одном
  уровне при разной длине текста.
* Выравнивание держится без фиксированных высот и без JS.
* В `CSS.md` своими словами: почему обычный вложенный grid так не умеет и что меняет `subgrid`.

## Подсказки

* Ищи: «CSS subgrid card layout», «grid-template-rows subgrid», «subgrid align card buttons».
* `subgrid` сегодня поддерживают все актуальные браузеры, на твоём Mac в свежем Chrome/Safari
  заработает. Если карточки не выравниваются — проверь, что карточка сама `display: grid` и
  занимает нужное число строк родителя.
